# Specs

Especificações técnicas de funcionalidades e componentes.
Descreve COMO algo deve funcionar (não o que será feito).

Exemplos de conteúdo:

- Especificação de componentes (formulário, CTAs)
- Contratos de integração (Formspree, GA4, Google Ads)
- Schemas de dados e eventos de tracking
- Critérios de aceite de features
